
# Url Shortner

A simple application that generate short Url




## Deployment

To deploy this project run

```bash
  make up_build
```


## Features

- Dockerize
- Cross platform


## Environment Variables

To run this project, you will need to add the following environment variables to your docker-compose file

`POSTGRES_USER`

`POSTGRES_PASSWORD`

`POSTGRES_DB`

`DSN`
